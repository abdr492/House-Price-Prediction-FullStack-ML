# 🏠 House Price Prediction System (Full Stack ML Project)

## 🚀 Overview

This project is an end-to-end **House Price Prediction System** built using:

* Machine Learning (Regression Models)
* FastAPI (Backend API)
* React.js (Frontend Dashboard)

It predicts house prices based on property features and provides a user-friendly dashboard for interaction.

---

## 🧠 Features

* 📊 Data preprocessing & feature engineering
* 🤖 Machine Learning model (Random Forest, Linear Regression)
* ⚡ FastAPI backend for real-time prediction
* 🎨 React dashboard UI
* 📈 Model evaluation (RMSE, MAE, R²)

---

## 🏗️ Architecture

Frontend (React) → Backend (FastAPI) → ML Model

## ⚙️ Installation

### 1. Clone repository

```
git clone https://github.com/YOUR_USERNAME/House-Price-Prediction-FullStack-ML.git
cd House-Price-Prediction-FullStack-ML
```

### 2. Backend setup

```
pip install -r requirements.txt
uvicorn app:app --reload
```

### 3. Frontend setup

```
cd frontend
npm install
npm start
```
